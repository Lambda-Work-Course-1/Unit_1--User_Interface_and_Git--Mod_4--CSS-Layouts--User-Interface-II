Hey! You are really getting around this little project of mine! For this page I designed a basic wireframe I thought would be easy to build out. For the challenge I used an auto generated color pallet! I used this color pallet. Because these colors are not CSS named colors I was able to take advantage of CSS variables! I am not sure how practical the variables were for this one page. Maybe on another larger project they would be more efficient.

Because I lack confidence in the design portion of web development the pallet made things much easier. This was a valuable lesson on looking for ways to bridge the gaps in knowledge/comfort and crush imposter syndrome.

I am not sure I amd thrilled with what I did with the colors... maybe in time I will grow to be more confidant in design.

Again, thanks for checking things out! If you have any feedback I am happy to hear it! I believe that we can only do better if our ideas and creations are challenged.
